## Current work

Building [TraceForward](https://github.com/bigelow/traceforward-mcp) — an MCP server that gives coding agents runtime context from OpenTelemetry backends before they touch your services. The idea: before an agent suggests restarting a service, it should know why that service is failing in your environment.

TraceForward is the public interface of a larger project called Meridian — a governed context substrate for AI agent collaboration across the software development lifecycle.
